Figure 6. ROC (Receiver Operating Characteristic) analysis of threshold assignment to the AIs-Deletions matrix of the mating signaling pathway.. From: Reverse Engineering a Signaling Network Using Alternative Inputs.

(A) Histogram of the AIs-Deletions matrix values for the mating pathway example. Values were placed in bins of size 10. The proportion of the total number of on and off values are indicated for each bin. (B) The ROC curve for the experimental data of the AIs-Deletions matrix of the mating signaling pathway. This curve represents the false positive rates (FPR) and true positive rates (TPR) for a range of threshold values. FPR is defined as FP (False positive)/(FP + TN (true negative)), and TPR is defined as TP (true positive)/(TP + FN (false negative)). Selected threshold values are shown on the curve.
