Figure 15. From: Integrating In Silico Resources to Map a Signaling Network.

Pathway Commons (accessed through Cytoscape). (A) BMP (bone morphogenetic protein) pathway – ‘File’ -> ‘Import’ -> ‘Network from Web Services…’ -> use ‘Smad1_human’ as input -> select ‘BMP receptor signaling’ from generated list -> double click. (B) SMAD1-centered Sub-network – type ‘smad1_human” into toolbar search box -> ‘Select’ -> ‘Nodes’ -> ‘First Neighbors of Selected Nodes’ -> click ‘Create new network from selected nodes, all edges’ icon on toolbar. (Some nodes were manually rearranged).
